❌ "Connection refused" error?
   → Terminal 1 crashed. Check for errors.
   → Restart: Ctrl+C, then re-run the uvicorn command

❌ "Port 8000 already in use"?
   → Kill existing process:
   → taskkill /F /IM python.exe
   → Then retry

❌ Streamlit keeps refreshing?
   → Normal on first run (downloads models)
   → Wait 2-3 minutes
   → Future runs are instant ⚡

❌ "No module named backend"?
   → Activate venv: .\.venv\Scripts\activate
   → Check you're in right directory:
   → cd c:\Users\asus\resumate-agentic-ai
